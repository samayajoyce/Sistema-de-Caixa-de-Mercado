# Sistema-de-Caixa-de-Mercado
Um projeto simples de lógica de programação desenvolvido para simular o funcionamento básico de um caixa de mercado
O sistema permite registrar produtos, calcular subtotal, aplicar desconto quando necessário e informar o troco do cliente. Esse tipo de projeto é útil para praticar estruturas fundamentais de programação, como variáveis, condicionais, laços de repetição e acumuladores.

🎯 Objetivo do projeto
Este projeto foi criado para treinar conceitos básicos de algoritmo e lógica de programação.

Principais objetivos:

    Praticar entrada e saída de dados.

    Trabalhar com variáveis e operadores aritméticos.

    Usar estruturas condicionais para regras de desconto e validação.

    Usar estruturas de repetição para registrar mais de um produto.

    Organizar um projeto de forma apresentável para portfólio no GitHub.

✨ Funcionalidades

    Cadastro de produtos no caixa.

    Leitura do nome do produto.

    Leitura do preço do produto.

    Leitura da quantidade comprada.

    Cálculo do subtotal de cada item.

    Soma do valor total da compra.

    Aplicação de desconto com base em uma regra definida.

    Leitura do valor pago pelo cliente.

    Cálculo do troco.

    Exibição do resumo final da compra.
